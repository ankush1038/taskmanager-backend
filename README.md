# Team Task Manager

A full-stack task management application where users can create tasks, track progress, and manage team workflows.

## Features

- User Registration & Login
- JWT Authentication
- Task Creation
- Task Status Tracking
- Dashboard Analytics
- Role-Based Backend Security
- PostgreSQL Database
- REST APIs
- Responsive UI

---

## Tech Stack

### Frontend
- React

- Axios
- React Router

### Backend
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- PostgreSQL

## Local Setup

### Backend

```bash
git clone <backend_repo_url>

cd taskmanager

./mvnw spring-boot:run
```

### Frontend

```bash
npm install

npm run dev
```

---

## Environment Variables

Backend requires:

```properties
DB_URL=
DB_USERNAME=
DB_PASSWORD=
JWT_SECRET=
```
